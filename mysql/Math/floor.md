### Mysql - floor()
floor()는 주어진 숫자를 내림 처리하여 반환한다.

**syntax**
```
floor(n)
```

**Argument**
Name|Description
-|-
N|A number


**Example**
```
floor(2.7);
floor(3.1);

select floor(2.7);
select floor(3.1);
```

**Result**
floor(2.7)|floor(3.1)
-|-
2|3
